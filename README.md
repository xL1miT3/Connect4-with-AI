# Connect4-mit-KI

Version 1:

Im Spiel Connect 4 (Vier gewinnt) gibt es einen Player vs KI-Modus. Die KI wurde mithilfe des Minimax-Algorithmus, Alpha-Beta-Pruning und einer Transpositionstabelle implementiert. Um den bestmöglichen Zug zu finden, bewertet die KI das Spielfeld, sobald eine bestimmte Suchtiefe erreicht wurde.

Version 2:

In dieser Version von Connect 4 (Vier gewinnt) wurde ein GUI mit Pygame implementiert. Um die Geschwindigkeit und Suchtiefe des Algorithmus erheblich zu verbessern, wurden Bitboards für die Spielfelder verwendet. Der Minimax-Algorithmus, Alpha-Beta-Pruning und eine Transpositionstabelle kommen zum Einsatz.
